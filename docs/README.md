# Ykamina.ru

## Локальная разработка

1. Поднимите PostgreSQL: `docker compose up -d postgres`.
2. Скопируйте `frontend/.env.example` в `frontend/.env.local`.
3. Запустите сайт: `cd frontend && npm run dev`.

## Данные

SQL-схема и демонстрационные данные лежат в `database/`. Повторный seed можно выполнить командой `npm run db:seed` из `frontend`.

## Импорт RealFlame

`https://realflame.ru/upload/partners.csv` — открытая партнёрская CSV-выгрузка. В ней есть название, артикул, цена, картинка, размеры и описание.

`https://realflame.ru/price/RealFlameStock.xlsx` содержит статусы доступности для Москвы и Санкт-Петербурга, а также оптовую и рекомендованную розничную цену. Точного числа единиц в файле нет, поэтому импортёр сохраняет фактические статусы (`Много`, `По запросу`) в `products.availability`, не подменяя их цифрами.

После запуска PostgreSQL выполните `npm run supplier:realflame:sync` из `frontend`. Команда загружает оба файла, определяет товар по артикулу, создаёт/обновляет карточку, цену, доступность и журнал запуска в `import_runs`. Для регулярного обновления добавьте эту команду в cron/планировщик раз в час. Не давайте доступ к импорт-эндпоинту из браузера — запуск должен оставаться серверной задачей.
